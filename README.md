# Laundry Service Hero Section

## Project Overview

This project is a simple landing page for a Laundry Service website built using HTML and CSS.

The webpage includes:

* A navigation bar with logo, menu items, and username button
* A hero section with heading, description, and call-to-action button
* An image section displaying a washing machine

## Technologies Used

* HTML5
* CSS3
* VS Code
* Git & GitHub

## Project Structure

```text
project-folder/
│
├── index.html
├── style.css
└── README.md
```

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder in VS Code.
3. Open `index.html`.
4. Run the file in browser:

   * Double-click `index.html`
     OR
   * Use Live Server in VS Code

## Features

* Simple and clean UI
* Responsive layout using CSS
* Navbar and hero section design
* Image and text alignment using CSS properties
